Building This Blog with Claude Code: My Hugo Experience

#hugo #claude-code #ai-development #blogging

I decided to start a blog and wanted to try something different - building it entirely with Claude Code as my development assistant. This post chronicles my experience creating this Hugo-based blog from scratch, entirely through conversation with AI.

Prerequisites

Make sure you have Hugo installed. If not, install it first:

# macOS with Homebrew
brew install hugo

# Verify installation
hugo version

Step 1: Create the Hugo Site

# Create new Hugo site
hugo new site blog
cd blog

# Initialize git repository
git init

Step 2: Install the Xmin Theme

# Add xmin theme as a git submodule
git submodule add https://github.com/yihui/hugo-xmin.git themes/xmin

# Create initial configuration
cat > hugo.toml << 'EOF'
baseURL = 'https://example.org/'
languageCode = 'en-uk'
title = 'Andrew Moores Blog'
theme = 'xmin'

[[menu.main]]
name = "Home"
url = "/"
weight = 1

[[menu.main]]
name = "Posts"
url = "/posts/"
weight = 2

[[menu.main]]
name = "About"
url = "/about/"
weight = 3

paginate = 5
paginatePath = "page"

[params]
customCSS = ["/css/custom-styles.css"]
# googleAnalytics = "G-XXXXXXXXXX"  # Replace with your Google Analytics 4 Measurement ID
EOF

Step 3: Create Content

# Create about page
cat > content/about.md << 'EOF'
---
title: "About"
---

Welcome to my blog! This is where I share thoughts on technology, development, and other topics that interest me.
EOF

# Create your first blog post
hugo new posts/installing-hugo.md
